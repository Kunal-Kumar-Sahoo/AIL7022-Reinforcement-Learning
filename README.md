### AIL7022 - Reinforcement Learning

This repository contains the asssignment and their solutions for the Reinforcement Learning course offered by IIT Delhi.

**Instructor:** [Prof. Raunak Bhattacharya](https://web.iitd.ac.in/~raunakbh/)
**Course Page:** https://web.iitd.ac.in/~raunakbh/deep-rl/fall25/coursepage.html

**Assignment Details:**
1. Assignment-1: **Policy Iteration and Value Iteration**
   - Value iteration and Policy iteration for stationary environment.
   - Value iteration and Policy iteration for non-stationary environment.
   - Modification of Value iteration algorithm for reducing the reducing the searchable state-space.
   - Value iteration and Policy iteration for finite-horizon MDPs.
   - Comparison of Value Iteration with Finite-horizon dynamic programming.
